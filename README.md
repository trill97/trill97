## 🚀 About Me

- 🛠️ I lead internal software development by day and run **Trillium Labs**, a web-development studio, on the side.
- 🌏 Based in **Sri Lanka**, building everything from internal business platforms to client web apps.
- 🧩 I work across the full stack — **Next.js / Laravel / Python** — and I'm into **AI-assisted development**, automation, and **fintech / algorithmic trading**.
